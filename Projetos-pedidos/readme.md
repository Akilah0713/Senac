# Projeto Pedidos 

O projeto Pedidos é uma aplicação criada para registrar, organizar e visualizar pedidos de forma rápida e prática. Ele permite adicionar novos pedidos e acompanhar tudo em um único painel de forma clara e intuitiva. O objetivo é praticar HTML, CSS, JavaScript e manipulação do DOM.

![Prévia do Projeto](/Imagens/prévia%20do%20projeto%20pédidos.png)

## Funcionalidades 
- Adicionar novos Pedidos
- Visualizar detalhes básicos de cada pedido
- Excluir pedidos que não são mais necessários
- Interface simples e fácil de usar
- Atualização instantânea da lista sem recarregar a págin
- Organização dos pedidos em cards ou lista
- Validação dos campos antes de cadastrar

## Tecnologias utilizadas 

- HTML
- CSS
- JavaScript
- Git e GitHub

## Como rodar
1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
2. Extraia a pasta
3. Abra o arquivo index.html no navegador

## Como clonar 
No terminal do VS Code digite:
    git clone https://github.com/Akilah0713/Senac.git
No GitHub Desktop:
    No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Akilah0713/Senac.git
