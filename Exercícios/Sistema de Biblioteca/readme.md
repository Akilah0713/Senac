# Projeto Sistema de Biblioteca 

Um sistema simples de cadastro e exibição de livros, desenvolvido com HTML, CSS e JavaScript, onde o usuário pode registrar obras e visualizá-las diretamente no navegador. O foco do projeto é treinar manipulação do DOM, arrays de objetos e lógica condicional.

![Prévia do Projeto](../Imagens%202/print%20projeto%20biblioteca.png)


## Funcionalidades 

- Formulário para cadastrar livros
- Armazena cada livro em um array de objetos no JS
- Lista todos os livros automaticamente no DOM
- Verifica o ano do livro e exibe uma tag:
- Clássico (vermelho) se ano < 2000
- Moderno (verde) se ano ≥ 2000
- Estilização completa com CSS externo
- Totalmente feito com JavaScript puro
