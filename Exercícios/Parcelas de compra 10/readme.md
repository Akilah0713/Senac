# Projeto Calculadora de Parcelas com Ajuste de Vencimentos

Um projeto em HTML, CSS e JavaScript que calcula o valor das parcelas de uma compra e exibe a data de vencimento de cada uma delas. O sistema também identifica quando o vencimento cai em um sábado ou domingo, exibindo um aviso de que o pagamento deverá ser adiado.

![Prévia do Projeto](../Imagens%202/print%20simulador%20de%20parcelas.png)


## Funcionalidades 

- Recebe o valor total da compra
- Recebe a quantidade de parcelas (1 a 12)
- Impede valores inválidos
- Calcula automaticamente o valor de cada parcela
- Gera as datas de vencimento mês a mês
- Exibe mensagem: "Vencimento adiado." caso caia no fim de semana
- Exibe tudo no HTML de forma organizada
