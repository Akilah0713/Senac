# Projeto Sistema de Descontos por Tipo de Cliente

Este projeto simula um sistema simples de descontos baseado no tipo de cliente.
O usuário informa o nome do produto, seu preço e o tipo de cliente (Comum, Estudante ou Assinante).
Com esses dados, o sistema calcula o valor do desconto e exibe todas as informações diretamente na página HTML.

## Funcionalidades 

- Captura o nome do produto
- Captura o preço do produto
- Captura o tipo de cliente
- Aplica automaticamente o percentual correto de desconto
- Calcula:
- Valor do desconto
- Preço final
- Exibe todas as informações no HTML
- Lógica simples e direta usando JavaScript puro
