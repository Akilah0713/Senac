# Projeto Simulador de Saque Bancário

Este projeto é uma simulação simples de saque bancário que utiliza HTML + JavaScript para solicitar dados do usuário, verificar saldo disponível e exibir um resumo completo da operação.

O objetivo do projeto é treinar lógica, estrutura condicional e manipulação do DOM.

## Funcionalidades 

- Captura o nome do cliente via prompt()
- Captura o saldo inicial
- Captura o valor do saque
- Verifica se há saldo suficiente
- Calcula o saldo final quando possível
- Exibe um resumo completo:
- Nome do cliente
- Saldo inicial
- Valor do saque
- Mensagem final (sucesso ou saldo insuficiente)
- Formatação de valores em duas casas decimais
- Manipulação de conteúdo via document.body.innerHTML
