# Projeto Calculadora de Parcelas com Ajuste de Vencimentos

Este projeto é um sistema simples desenvolvido com HTML, CSS e JavaScript que simula o funcionamento de um caixa de supermercado.
O usuário informa três produtos diferentes e o sistema calcula automaticamente os subtotais, o total, verifica descontos e exibe o resultado final.

![Prévia do Projeto](../Imagens%202/print%20simulador%20de%20parcelas.png)


## Funcionalidades 

- Cadastro de 3 produtos com nome, categoria, quantidade e preço
- Cálculo automático do subtotal de cada item
- Soma total da compra
- Verificação automática de desconto por categoria
- Verificação de desconto por valor total
- Descontos acumulativos (5% + 10%)
