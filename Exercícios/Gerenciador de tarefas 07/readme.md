# Projeto Gerenciador de Tarefas

Um projeto simples em HTML, CSS e JavaScript desenvolvido para cadastrar tarefas, organizá-las por prioridade e exibi-las automaticamente na tela. Este projeto é ideal para praticar formulário, manipulação do DOM, arrays, objetos e lógica de programação.

![Prévia do Projeto](../Imagens%202/projeto%2001.png)


## Funcionalidades 

- Formulário com:
- Campo para título
- Campo para descrição
- Select com prioridades (Alta, Média, Baixa)
- Botão Adicionar
- Criação automática de objetos representando cada tarefa
- Armazenamento das tarefas em um array
- Exibição de todas as tarefas na tela
- Cores diferentes conforme a prioridade
- Interface simples e prática
- Feito totalmente com JavaScript puro

## Tecnologias utilizadas 

- HTML
- CSS
- JavaScript
- Git e GitHub

## Como rodar
1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
2. Extraia a pasta
3. Abra o arquivo index.html no navegador

## Como clonar 
No terminal do VS Code digite:
    git clone https://github.com/Akilah0713/Senac.git
No GitHub Desktop:
    No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Akilah0713/Senac.git
