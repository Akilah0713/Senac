# Projeto Sistema de Reserva de Viagens

Este projeto é um sistema simples de cadastro e validação de reservas de viagem, desenvolvido com HTML, CSS e JavaScript.
O usuário pode registrar uma reserva informando seu nome, o destino e a data da viagem.
O sistema salva cada reserva em um array de objetos e exibe todas as reservas cadastradas diretamente no HTML, com validação visual das datas.

![Prévia do Projeto](../Imagens%202/print%20projeto%20reserva%20de%20viagens.png)


## Funcionalidades 

- Formulário com três campos: nome, data e destino
- Cada reserva é salva em um array de objetos
- Exibição automática das reservas no HTML
- Validação da data em comparação com a data atual
- Cores e mensagens diferentes para cada situação:
- Vermelho → Data inválida
- Amarelo → A viagem é hoje
- Verde → Viagem confirmada
- Estilização feita com CSS externo
- Interface simples e intuitiva
- Sistema funcional feito apenas com JavaScript puro
