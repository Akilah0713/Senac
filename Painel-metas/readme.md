# Projeto Painel metas

O Painel de Metas é uma aplicação criada para ajudar o usuário a organizar e visualizar suas metas de forma clara e objetiva. Nele é possível adicionar metas, marcá-las como concluídas e acompanhar o progresso de maneira prática. O objetivo é praticar HTML, CSS, JavaScript e manipulação do DOM.

![Prévia do Projeto](/Imagens/prévia%20do%20projeto%20painel%20metas.png)

## Funcionalidades 
- dicionar novas metas à lista
- Marcar metas como concluídas
- Excluir metas que não são mais necessárias
- Visualizar todas as metas em um só lugar
- Atualização instantânea sem recarregar a página
- Design limpo e agradável

## Tecnologias utilizadas 

- HTML
- CSS
- JavaScript
- Git e GitHub

## Como rodar
1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
2. Extraia a pasta
3. Abra o arquivo index.html no navegador

## Como clonar 
No terminal do VS Code digite:
    git clone https://github.com/Akilah0713/Senac.git
No GitHub Desktop:
    No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Akilah0713/Senac.git
