# Projeto Frases Motivacionais

O Gerador de Frases Motivacionais é uma aplicação que exibe frases inspiradoras de forma aleatória. Toda vez que o usuário clica no botão, uma nova mensagem motivacional aparece para incentivar o dia e trazer mais positividade. O objetivo é praticar HTML, CSS, JavaScript e manipulação do DOM.

![Prévia do Projeto](/Imagens/prévia%20do%20projeto%20das%20frases%20motivacionais.png)

## Funcionalidades 
- Gerar uma nova frase motivacional ao clicar em um botão
- Selecionar frases aleatórias de uma lista pré-definida
- Exibir a frase na tela em tempo real
- Atualização instantânea sem recarregar a página
- Possibilidade de trocar a frase quantas vezes quiser
- Design limpo e agradável

## Tecnologias utilizadas 

- HTML
- CSS
- JavaScript
- Git e GitHub

## Como rodar
1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
2. Extraia a pasta
3. Abra o arquivo index.html no navegador

## Como clonar 
No terminal do VS Code digite:
    git clone https://github.com/Akilah0713/Senac.git
No GitHub Desktop:
    No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Akilah0713/Senac.git
