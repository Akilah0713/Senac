# Projeto Gerador de dicas de estudo

O Gerador de Dicas de Estudo exibe dicas aleatórias para ajudar quem está estudando a melhorar sua organização, foco e produtividade. A cada clique, uma nova dica aparece para orientar e incentivar os estudos. O objetivo é praticar HTML, CSS, JavaScript e manipulação do DOM.

![Prévia do Projeto](/Imagens/prévia%20do%20projeto%20de%20dicas%20de%20estudo.png)

## Funcionalidades 
- Gerar uma dica de estudo aleatória com apenas um clique
- Exibir dicas diferentes a cada interação
- Atualização instantânea sem recarregar a página
- Botão para gerar novas dicas
- Possibilidade de exibir dicas relacionadas a foco, organização, memorização e produtividade

## Tecnologias utilizadas 

- HTML
- CSS
- JavaScript
- Git e GitHub

## Como rodar
1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
2. Extraia a pasta
3. Abra o arquivo index.html no navegador

## Como clonar 
No terminal do VS Code digite:
    git clone https://github.com/Akilah0713/Senac.git
No GitHub Desktop:
    No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Akilah0713/Senac.git
